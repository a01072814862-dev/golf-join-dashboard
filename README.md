# 골프조인 호스트 관리 대시보드

PC에서 호스트가 핫딜과 라운드를 관리할 수 있는 웹 기반 관리 대시보드입니다.

## 기능

### 1. 대시보드
- 활성 핫딜 수
- 대기 중인 구매요청 수
- 완료된 거래 수
- 총 할인액
- 최근 구매요청 목록

### 2. 핫딜 관리
- 새 핫딜 등록
- 등록된 핫딜 목록 조회
- 핫딜 수정 (예정)
- 핫딜 삭제
- 할인율 자동 계산

### 3. 구매요청 관리
- 모든 구매요청 조회
- 상태별 필터링 (대기 중, 확정됨, 취소됨)
- 요청 수락/거절
- 요청자 연락처 표시

### 4. 라운드 관리
- 라운드 등록 (예정)
- 라운드 목록 조회 (예정)
- 라운드 수정/삭제 (예정)

### 5. 설정
- 호스트 정보 수정
- 이메일, 연락처 관리

## 접근 방법

### 로컬 개발 환경
```bash
# 웹 서버 실행 (Python)
cd /home/ubuntu/golf-join/web
python3 -m http.server 8000

# 브라우저에서 접속
http://localhost:8000/pages/login.html
```

### 테스트 계정
- **이메일**: host@example.com
- **비밀번호**: password123

## 데이터 저장

현재는 브라우저의 localStorage를 사용하여 데이터를 저장합니다. 

### 저장되는 데이터
- `golf_join_host_data`: 호스트 정보
- `golf_join_hot_deals`: 핫딜 목록
- `golf_join_requests`: 구매요청 목록

## 사용 방법

### 1. 핫딜 등록
1. "핫딜 관리" 탭 클릭
2. "+ 새 핫딜 등록" 버튼 클릭
3. 골프장 정보 입력
4. "등록" 버튼 클릭

### 2. 구매요청 관리
1. "구매요청 관리" 탭 클릭
2. 요청 목록 확인
3. "수락" 또는 "거절" 버튼 클릭
4. 요청자 연락처로 연락

### 3. 설정 수정
1. "설정" 탭 클릭
2. 호스트 정보 수정
3. "저장" 버튼 클릭

## 문제 해결

### 데이터가 저장되지 않음
- 브라우저의 localStorage가 활성화되어 있는지 확인
- 개인정보 보호 모드에서는 localStorage가 작동하지 않을 수 있음

### 로그인이 안 됨
- 테스트 계정 정보 확인: host@example.com / password123
- 브라우저 캐시 삭제 후 다시 시도
